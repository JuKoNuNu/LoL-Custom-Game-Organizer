# LoL-Custom-Game-Organizer!
해당 웹 사이트는 게임 "리그 오브 레전드"의 API를 활용하여
기존 서비스에는 없는 내전용 서비스입니다.

기존 OP.GG 같은 웹 사이트에는 전적 검색 기능만 있으나
해당 서비스는 사설 전적을 검색해 DB에 저장한 후에
전적, 티어에 기반해 점수로 나눈뒤 인원에 맞게 팀을 정합니다.
 또한 랜덤, 라인 랜덤, 고정 및 분리 인원 기능으로 사용자의
요구에 맞춰 팀을 정한 뒤 디스코드 웹 훅으로 전송하는 기능입니다.

또한 AI를 활용하는 방법을 익히기 위해 CLAUDE CLI를 연동하여

프롬포트를 짠 후, 제가 원하는 로직을 구현할 수 있게 하였습니다.
실제로 모든 구성과 구조를 정한 뒤 CLAUDE를 사용해 짧은 기간만에 해당 기능을 구현할 수 있었습니다.
